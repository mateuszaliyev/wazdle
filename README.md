# Wążdle

## Installation

Clone the repository.

```bash
git clone https://github.com/mateuszaliyev/wazdle.git

# or

git clone git@github.com:mateuszaliyev/wazdle.git
```

Go to the project directory.

```bash
cd wazdle
```

Install [PyQt6](https://pypi.org/project/PyQt6/).

```bash
pip install PyQt6
```

## Usage

Run application.

```
python ./src/main.py
```
